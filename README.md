# Data Cleaning & Reporting Automation

## Overview

This project automates data cleaning and reporting using Python and Pandas. It processes raw datasets by handling missing values, removing duplicate records, and standardizing inconsistent data formats.

## Features

* Missing value handling
* Duplicate record removal
* Data standardization
* Automated report generation
* Export cleaned data to Excel

## Technologies Used

* Python
* Pandas
* OpenPyXL

## Output

* Cleaned dataset
* Automated summary report
* Excel export file
